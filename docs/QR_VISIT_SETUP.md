# Thiet lap QR va thong ke luot truy cap POI

Chay SQL sau trong Supabase SQL Editor.

```sql
create table if not exists public."PoiVisit" (
  id bigint generated by default as identity primary key,
  poi_id bigint not null references public.poi(id) on delete cascade,
  source text not null default 'qr_web',
  created_at timestamptz not null default now()
);

create index if not exists idx_poi_visit_poi_id_created_at
  on public."PoiVisit" (poi_id, created_at desc);

alter table public."PoiVisit" enable row level security;

-- Public duoc ghi nhan luot quet tu trang /map/scan.html
create policy "poi_visit_insert_public"
  on public."PoiVisit"
  for insert
  to anon, authenticated
  with check (poi_id is not null);

-- Admin da dang nhap co the doc de hien thi dashboard
create policy "poi_visit_select_authenticated"
  on public."PoiVisit"
  for select
  to authenticated
  using (true);
```

Ghi chu:
- Dashboard admin can user authenticated de doc du lieu PoiVisit.
- Link QR cua tung POI co dang /map/scan.html?poi=ID.
- Moi lan mo link QR se tao 1 ban ghi moi trong PoiVisit.
